# customer_behaviour_anaysis
data analytics project showcasing customer behaviour analysis using python,sql and powerbi

📊 Data Analytics Project
🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating actionable insights through visualization and reporting. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, and dashboard creation.

📁 Dataset

Source: Kaggle (Customer Shopping Behavior Dataset)

Format: CSV 

Description: Contains customer behavior and transaction-related data used for analysis.

🛠️ Tools & Technologies

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL / MySQL / SQL Server)

Power BI (Dashboard & Visualization)

Gamma (Presentation creation)

Jupyter Notebook / VS Code

⚙️ Project Steps
1. Data Loading

Imported dataset using Python (Pandas)

Checked structure, data types, and basic info

2. Exploratory Data Analysis (EDA)

Identified trends, patterns, and anomalies

Used visualizations like bar charts, histograms, and heatmaps

3. Data Cleaning

Handled missing values

Removed duplicates

Corrected data types

Standardized inconsistent values

4. SQL Analysis

Loaded cleaned data into database

Performed queries to extract insights:

Aggregations (SUM, AVG, COUNT)

Filtering and grouping

Business-related KPIs (e.g., discount rates, top products)

5. Dashboard Creation (Power BI)

Built interactive dashboard

Included KPIs, filters, and visuals

Focused on business insights and usability

6. Reporting & Presentation

Created a structured report of findings

Designed a presentation using Gamma

Highlighted key insights and recommendations

📊 Dashboard

Interactive Power BI dashboard showcasing:

Sales trends

Customer behavior

Product performance

Discount analysis

📈 Key Results

Identified top-performing products

Analyzed discount impact on sales

Discovered customer purchasing patterns

Generated actionable business insights

Open Jupyter Notebook / VS Code

Refresh data if needed
